Print

Команда print використовується для виведення тексту(символів).

Текст для виведення береться в лапки в дужках.

Також можна виводити значення змінних. Для початку потрібно прив'язати цій змінній якийсь текст, символ чи число. Для цього використаємо таку команду: Назва змінної="Якийсь текст" або Назва змінної=Якесь число. Зверніть увагу, що при призначенні змінній числа ми не беремо його в лапки навідмінно від текста. Потім, щоб вивести значення використаємо функцію print(), при цьому ми пишемо назву змінної в душках, не биручи її в лапки.

Типи даних

При програмуванні мовою Python використовується достатньо велика кількість типів даних. Але є основні, такі як:

  • boolean — логічнтй тип
  • integer — ціле число
  • float — дійсне число
  • string — рядок

Також можна перевірити тип даних, які прив'язані до змінних за допомогою функції print(), а саме:

Іноді потрібно змінити тип даних змінної, наприклад з рядка на число, щоб надалі виконувати арифметичні дії над числом. Для цього використаємо таку команду: Назва змінної=Новий тип(Назва змінної)

Арифметичні дії

Основними арифметичними діями в мові програмування Python є: +, -, *, /, **, //, %.

  • + — використовується для додавання чисел або додавання одного рядка до іншого.
  • - — використовується для віднімання чисел.
  • * — використовується для множення чисел або для повторення рядка.
  • / — використовується для ділення чисел.
  • ** — використовується для піднесення числа до степеня.
  • // — використовується для ділення чисел без остачі.
  • % — використовується для визначення остачі від ділення чисел.

Цикли

В мові Python часть використовують цикли різних типів. Цикли for та while. Цикл має таку будову: умова, тіло.

Цикл while (з англ. - доки) виконує тіло циклу до того часу, доки умова циклу виконується, а також цикл for.

Використовуючи команди if (якщо), else (інакше) та elif(інакше якщо), можна робити розгалудження.

Оператори continue та break

Оператор continue починає наступний прохід циклу, минаючи тіло циклу, що залишилося.

Оператор break достроково перериває цикл.
